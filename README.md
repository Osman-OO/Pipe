## ï¿½ Usage Examples

### 1. Sales Performance Analysis
```bash
# Comprehensive sales analysis with forecasting
python -m pipe.analytics \
  --dataset data/sales_data.csv \
  --analysis sales_performance \
  --forecast 12 \
  --output reports/sales_analysis.html

# Generate executive dashboard
python -m pipe.dashboard \
  --config configs/sales_dashboard.yaml \
  --port 8050
```

### 2. Customer Segmentation & Churn Prediction
```bash
# RFM analysis and customer clustering
python -m pipe.ml \
  --model customer_segmentation \
  --data data/customer_transactions.csv \
  --features recency,frequency,monetary \
  --clusters 5

# Churn prediction model
python -m pipe.ml \
  --model churn_prediction \
  --algorithm random_forest \
  --train data/customer_features.csv \
  --target churned \
  --validate --save-model
```

### 3. Financial Risk Analysis
```bash
# Portfolio risk assessment
python -m pipe.finance \
  --analysis portfolio_risk \
  --data data/stock_prices.csv \
  --var-confidence 0.95 \
  --monte-carlo 10000

# Fraud detection pipeline
python -m pipe.ml \
  --model fraud_detection \
  --algorithm isolation_forest \
  --data data/transactions.csv \
  --anomaly-threshold 0.1
```

### 4. A/B Testing Framework
```bash
# Statistical significance testing
python -m pipe.experiments \
  --test ab_test \
  --control data/control_group.csv \
  --treatment data/treatment_group.csv \
  --metric conversion_rate \
  --confidence 0.95
```
